## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18+)
- Python (3.9+)
- MongoDB

### Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/hackbuddy.git
cd hackbuddy
```

2. Install Frontend Dependencies
```bash
cd frontend
npm install
```

3. Install Backend Dependencies
```bash
cd ../backend
npm install
```

4. Install ML Service Dependencies
```bash
cd ../ml-service
pip install -r requirements.txt
```

5. Set Up Environment Variables
- Create `.env` files in frontend, backend, and ml-service directories
- Add necessary configuration (MongoDB Atlas URL, Secret keys, Github Token and Gemini API Key)

### Running the Application

1. Start Backend Service
```bash
cd backend
npm start
```

2. Start Frontend
```bash
cd frontend
npm run dev
```
